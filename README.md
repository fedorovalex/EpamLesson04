Задание ООП
- Создайте класс Человек. Его атрибуты - имя, дата рождения, номер документа удостоверяющего личность. Люди должны "сравниваться". Человек "больше" если его возраст больше. Если возраст одинаковый - у кого "больше" номер документа.

- Создайте 5 человек, поместите их в массив. Выведите его в порядке возрастания возраста, потом в алфавитном порядке по именам. (утилита Arrays, интерфейсы Comparator, Comparable)
